# Student Portfolio Website

## Project Overview
This project was developed as part of COS 106 – Introduction to Web Technologies.
It is a responsive personal portfolio and academic management website designed
to showcase student information, skills, projects, and achievements.

## Features

- Responsive design for desktop and mobile devices
- Interactive navigation menu
- Animated statistics section
- Student profile presentation
- Project showcase
- Contact form
- Clean and modern user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
